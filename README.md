# Hayat: AI-Powered Emergency Response Platform

Hayat is an intelligent emergency response platform that uses Artificial Intelligence to analyze emergency reports, classify incidents, and improve communication between users and emergency agencies.

## Project Overview

Hayat provides a unified platform that connects citizens with emergency services by allowing users to submit reports using text, images, voice recordings, and location data.

The AI system analyzes the submitted information to:
- Detect the emergency type
- Classify incident severity
- Extract important details
- Recommend the appropriate emergency agency
- Generate structured emergency reports

## Key Features

- 📍 GPS-based emergency location detection
- 🎤 Voice-to-text emergency reporting
- 📷 Image-based incident analysis
- 🤖 AI-powered incident classification
- 🚨 Emergency priority detection
- 🌐 Bilingual report generation (Arabic & English)
- 👮 Officer dashboard for reviewing reports

## AI Technologies Used

- Whisper Speech-to-Text
- TF-IDF + Machine Learning classifiers
- Naive Bayes / SVM models
- Hugging Face Transformers
- Computer Vision models (CNN, YOLO, CLIP)

## System Workflow

1. User submits an emergency report.
2. The system collects text, image, voice, and location information.
3. AI analyzes the input and identifies the incident category.
4. The report is prioritized and forwarded to the suitable agency.
5. Emergency officers review and manage the report.
